Personal Library (Ongoing)

This library management web application built using HTML, CSS, and JavaScript. Users can fill out a form with book details such as author, number of pages, and whether the book is read or unread.The submitted book information is added to a collection of books displayed on the page. Users can remove books from the collection by clicking a delete button associated with each book entry. The collection of books can be sorted based on whether they are read or unread.
